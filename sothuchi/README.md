# 💰 Sổ Thu Chi - Quản lý chi tiêu thông minh

Ứng dụng quản lý thu chi cá nhân với AI tích hợp (Claude AI).

## Tính năng

- ✅ Ghi thu chi nhanh với 12 danh mục
- ✨ AI tự động phân loại giao dịch theo ghi chú
- 📊 Báo cáo biểu đồ theo tháng
- 💼 Quản lý ngân sách theo danh mục
- 🤖 Trợ lý AI tư vấn tài chính (Claude AI)
- 💸 Quản lý vay nợ
- 🔍 Tìm kiếm và lọc giao dịch
- 📈 Theo dõi tỷ lệ tiết kiệm
- 🔔 Cảnh báo vượt ngân sách & sắp đến hạn nợ

## Cài đặt local

```bash
npm install
npm run dev
```

## Deploy lên Vercel

1. Push code lên GitHub
2. Vào vercel.com → New Project → Import repo này
3. Nhấn Deploy → Xong!

## Cài app lên điện thoại (PWA)

1. Mở link Vercel trên trình duyệt điện thoại
2. Nhấn "Chia sẻ" → "Thêm vào màn hình chính"
3. App sẽ xuất hiện như app thật trên điện thoại
